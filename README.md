# 🎬 CineMatch — Movie Recommendation Engine

A Flask-based web app that recommends Bollywood & Hollywood movies based on your **Genre** and **Mood**.

---

## ✨ Features

- 🎭 Filter by **Genre** (Action, Romance, Horror, etc.)
- 😊 Filter by **Mood** (Happy, Sad, Thrilling, Motivational, etc.)
- 🌍 Choose between **Bollywood**, **Hollywood**, or **Both**
- 🎨 Beautiful dark cinematic UI
- ⚡ Fast Flask backend

---

## 🚀 How to Run

```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/movie-recommender.git
cd movie-recommender

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the app
python app.py

# 4. Open browser
# Go to: http://localhost:5000
```
